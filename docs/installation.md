# Installation

### deno.land

```ts
import nhttp from "https://deno.land/x/nhttp@1.3.10/mod.ts";
```

### nest.land

```ts
import nhttp from "https://x.nest.land/nhttp@1.3.10/mod.ts";
```

### npm/yarn

```bash
npm i nhttp-land

// or

yarn add nhttp-land
```

```ts
// module
import nhttp from "nhttp-land";

// commonjs
const nhttp = require("nhttp-land").default;
```
