# Basic Template
This is basic template starter nhttp.

### Directory Structure
```md
├── router
│   ├── home_router.ts
│   └── user_router.ts
├── deno.json
├── main.ts
```

### Command
#### Deno
```bash
deno task dev
deno task start
```
#### Npm
```bash
npm run dev
npm run start
```
#### Bun
```bash
bun run dev
bun run start
```
