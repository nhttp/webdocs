Intro

NHttp a just native HTTP/2 micro framework for Deno. so hot ðŸš€

Note: Deno native HTTP/2 Hyper requires Deno version 1.9.0 or higher.

Features#

  • HTTP/2 support.
  • Middleware support.
  • Router support.
  • Includes body parser (jsonBody, urlencodedBody).
  • No third party modules and no std/lib by default.
  • Easy deploy to Deno Deploy and Cloudflare Workers.
